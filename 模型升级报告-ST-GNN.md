# 风速误差校正模型说明——时空图神经网络（ST-GNN）

## 1. 背景

早期方案采用4层全连接神经网络（FCN），仅以同时刻邻近风机的观测值作为输入特征，忽略了风速的时间序列依赖关系和风机间的空间拓扑结构，预测精度有限。本次将模型升级为时空图神经网络（Spatial-Temporal GNN），在空间和时间两个维度上对多风机系统进行联合建模。

## 2. 模型架构

整体分为三个模块：时序特征编码、空间图卷积聚合、预测输出。

```
输入: [n_nodes, input_dim]
  |
  v
输入投影层 (Dense, hidden_dim=64)
  |
  v
4层图卷积（GraphConvLayer × 4）
  每层: A·H → Linear → ReLU → Dropout
  每隔2层添加残差连接
  |
  v
节点注意力加权
  att = Sigmoid(Dense(x))
  x = x * att
  |
  v
输出MLP: hidden_dim → 32 → 16 → 1
  |
  v
输出: 目标风机预测风速
```

**邻接矩阵**：基于风机经纬度坐标，以高斯核计算节点间相似度，按行归一化：

$$A_{ij} = \exp\left(-\frac{d_{ij}^2}{2\sigma^2}\right), \quad \hat{A} = D^{-1}A$$

其中 $d_{ij}$ 为风机 $i$ 与 $j$ 之间的地理距离，$\sigma$ 取所有距离的中位数。

## 3. 与旧模型对比

| 模块 | FCN（旧） | ST-GNN（新） |
|------|-----------|-------------|
| 时序建模 | 无（单时刻输入） | 全连接时序编码器 |
| 空间建模 | 特征直接拼接 | 4层图卷积，消息传递 |
| 注意力机制 | 无 | 节点级自适应加权 |
| 残差连接 | 无 | 每2层一次跳跃连接 |
| 参数量 | ~10k | ~50k |
| 正则化 | BatchNorm | BatchNorm + Dropout(0.2) |

## 4. 候选风机选取策略

对每段缺失数据，采用三路并行方式选取参考风机：

- **DTW相似度**：基于动态时间规整，选取历史风速序列形态最相似的风机
- **Pearson相关**：选取与目标风机线性相关性最高的风机  
- **地理距离**：选取空间上最近的风机

三路各独立训练预测，最终取均值作为输出（集成策略）。

## 5. 实验配置

- 框架：Huawei MindSpore 2.8.0，CPU模式
- 数据：108台风机，时间分辨率1小时
- 历史窗口：24小时
- 每路候选风机数：4台
- 训练轮数：10 epochs（每个缺失块单独训练）
- 优化器：Adam，学习率0.001

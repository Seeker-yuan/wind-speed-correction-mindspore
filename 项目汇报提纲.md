---

## 3️⃣ **技术方案**（2-3分钟）

### 核心创新点：

#### ✅ **1. 使用华为MindSpore深度学习框架**
```
原方案：TensorFlow图神经网络
新方案：MindSpore全连接神经网络
```

#### ✅ **2. 模型架构设计**
```
输入层 (4个邻近风机的风速数据)
    ↓
全连接层 (64神经元) + BatchNorm + ReLU + Dropout
    ↓
全连接层 (64神经元) + BatchNorm + ReLU + Dropout
    ↓
全连接层 (32神经元) + BatchNorm + ReLU
    ↓
输出层 (预测的目标风机风速)
```

**关键技术**：
- BatchNormalization：提升训练稳定性
- Dropout：防止过拟合
- Adam优化器：自适应学习率

#### ✅ **3. 三路特征融合（保留原有优势）**
1. **DTW时序相似度** - 找出时序模式相似的风机
2. **Pearson相关系数** - 找出数据线性相关的风机
3. **地理位置距离** - 找出空间位置接近的风机

**集成策略**：三路预测结果加权平均

**Q: 为什么用全连接网络而不是图神经网络？**
A: 全连接网络更容易实现和调试，在当前数据规模下性能已经满足需求。如果需要，可以扩展为图神经网络来建模风机之间的空间拓扑关系。

**Q: MindSpore相比TensorFlow有什么优势？**
A: MindSpore是华为自主框架，支持国产化；性能优化针对昇腾硬件；更轻量级；支持自动并行和混合精度训练。

**Q: 如何保证预测精度？**
A: 采用三路特征融合（DTW+Pearson+位置），集成学习提升鲁棒性；使用24小时历史窗口；选择4台最相似风机作为参考。

**Q: 能否对接华为云服务？**
